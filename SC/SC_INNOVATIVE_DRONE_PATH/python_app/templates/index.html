PSO-Based UAV Path Planning

Compare PSO (particle swarm on continuous waypoints), A*, and Dijkstra on a 2D grid. All pathfinding runs on a Python Flask server; this page visualizes results, plays search expansion, and shows Matplotlib comparison charts. Paths are forced to start at S and end at E; polyline animation uses the true SVG path length (no truncated endpoint illusion).
